# Flyrank ML Internship Starter Project

This repository contains the weekly notebook submissions and machine learning pipelines for the Flyrank ML engineering internship. The focus of this project is analyzing anonymized search data, building predictive models to identify declining content, and breaking down complex data patterns into human-readable decision rules.

## Project Overview

- **Week 1 (Discovery):** Explored 30,000 rows of anonymized search data to perform exploratory data analysis (EDA) and challenge common SEO myths regarding keyword volume, length, and traffic.
- **Week 2 (Readable Modeling):** Built hand-written heuristics and paired them against a depth-constrained Decision Tree classifier to create highly interpretable, actionable business logic while avoiding data leakage.

## Tech Stack & Core Libraries

- **Language:** Python
- **Data Analysis:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn (Decision Tree Classifiers, Random Forests)
